# Abdiqani Hassan Personal Website 

Welcome to my personal website that I created using Typescript, React, Next.js, and Tailwind CSS.

- This website contains information about my programming and different projects that I have completed during my own personal time and as a Computer Science student at Carleton University, and other additional hobbies that I find enjoyable. 


### How to Run 
- First, clone the repository and then input the following commands:
```bash
npm install
npm run dev
```
- After running `npm run dev`, the application will run locally!
